# Stock Project - 股票龙虎榜

基于 Vue 3 + Vite 的股票龙虎榜应用
支持 Node 16+
```

## 安装依赖

```bash
npm install
```

## 开发

```bash
npm run dev
```

启动后访问 `http://localhost:3000`

## 构建

```bash
npm run build
```

## 预览构建结果
